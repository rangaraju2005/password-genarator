

 Password Generator in Python

A simple Python program that generates a strong random password using letters, numbers, and special characters based on user-defined length.

---

 Features

* Generates secure random passwords
* Uses:

  * Uppercase & lowercase letters
  * Numbers
  * Special characters
* User-friendly (takes input from the user)
* Beginner-friendly Python project

---

Technologies Used

* Python
* Built-in libraries:

  * `random`
  * `string`
